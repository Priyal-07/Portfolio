Online Bookstore

A Spring Boot web app for exploring, managing, and ordering from a catalog of books. Features user accounts, admin controls, and role-based access.

User Flow
  • Sign In & Sign Up: Users create an account or log in to existing one to access the bookstore.
    • New users provide their details, which are securely stored in the database. Existing users log in with authentication handled by Spring Security.
Tech Stack
  • Java and Spring Boot: Core backend framework.
  • MVC Architecture: For separation of concerns.
  • Java Beans: To encapsulate business logic.
  • Database Management: In-memory H2 database.
  • Security: Role-based access control.
  • Thymeleaf: Server-side rendering of web pages.
  • HTML & CSS: To design and structure web page.
