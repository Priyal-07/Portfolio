Online Bookstore

This is a Java-based web application built with the Spring Boot framework, allowing users to explore a catalog of books and place orders. Users can create accounts, log in, and manage their cart. The application also includes an admin role, enabling administrators to add or remove books from the catalog.

Technical Skills
  • Java & Spring Boot: Developed a robust application utilizing Spring Boot, Spring MVC, and Spring Security frameworks.
  • MVC Architecture: Designed and implemented the Model-View-Controller architecture for separation of concerns.
  • Java Beans: Created and managed Java Beans to encapsulate business logic.
  • Thymeleaf: Utilized Thymeleaf for server-side rendering of web pages.
  • Database Management: Implemented H2 in-memory database with custom SQL queries for efficient data management.
  • Security: Integrated role-based access control with Spring Security, custom UserDetailsService, and SecurityConfig.
  • Spring Data JPA: Leveraged Spring Data JPA in controllers for streamlined database interactions.
  • HTML & CSS: Designed and structured web content using HTML and CSS for a user-friendly interface.
Soft Skills

  • Adaptability: Quickly adapted to new technologies and frameworks as needed.
